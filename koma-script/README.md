## TColorBox
### Theorems (dutch)
``tex
\begin{THEOREM}{TITLE}{REFNAME}
    TEXT
\end{THEOREM}
``
Note: `TITLE` and `REFNAME` are optional.

#### Theorem name and refname-prefix
- Definitie: `def`
- Axioma: `ax`
- Eigenschap: `prop`
- Lemma: `lem`
- Stelling: `stel`
- Bewering: `bwr`
- Propositie: `prop`
- Gevolg: `gev`
- Bewijs: `bew`
- Voorbeeld: `vb`
- Toepassing: `tp`
- Oefening: `oef`

### Boxes
``tex
\begin{THEOREM}{TITLE}
    TEXT
\end{THEOREM}
``
Note: `TITLE` is optional.

#### Box name
- Oplossing
- Antwoord
- Uitwerking
- Opmerking
- Hint
- Herinner
- Intermezzo
- Notatie
- Conventie
- Vraag
